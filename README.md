## üöÄ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn
- Angular CLI

### Installation

1. Clone the repository
```bash
git clone git@github.com:reachvivek/yoga-landing-page.git
cd yoga-landing-page
```

2. Install dependencies
```bash
npm install
```

3. Run development server
```bash
ng serve
```

4. Build for production
```bash
ng build
```
